Project P — 001 · Featured

DNA Amplification
Automation System

A fully closed-loop mechatronic platform that automates LAMP and PCR DNA amplification end-to-end — from reagent routing to colorimetric readout.

Final Year Project Dublin City University · 2025 Mechatronic Build
01

Overview

LAMP and PCR are biological techniques used to amplify trace amounts of DNA into quantities large enough to detect. Both demand careful liquid handling, precise thermal cycling, and consistent timing — traditionally performed by hand or on expensive lab equipment.

This project consolidates that workflow into a single, closed-loop machine. Reagents live on a disposable cartridge disk; the machine spins the disk, routes liquids through microfluidic channels via centrifugal force, applies controlled heat for amplification, and reads the result optically — all under autonomous orchestration by a master control node.

The system is modular by design. Each function — drive, thermal, vision, control — runs on its own node and communicates with the master over a shared messaging bus, allowing the disk type and instruction sequence to drive the entire process.

02

System Architecture

N — 01
Master Control Node

Maintains a registry of disk types and the instruction sequence each one requires. Orchestrates inter-node messaging, sequencing, and precise timing across the full process. Effectively the brain that turns a high-level "run this protocol" command into coordinated low-level actions.

N — 02
Primary Drive

A main motor spins an offset disk to generate centrifugal forces that drive liquid movement through the microfluidic channels. The geometry of the offset disk converts rotational speed into directional fluid motion without contact-based pumping.

N — 03
PID-Controlled Force Vector

A secondary motor under closed-loop PID control dynamically alters the direction of centrifugal force, routing reagents through different stages of the assay. This is the key control problem: keeping the force vector aligned to the right channel at the right time.

N — 04
Thermal Subsystem

A controlled heater manages thermal cycling for DNA amplification. LAMP runs isothermally, PCR requires step changes — the controller handles both profiles under the master node's instruction.

N — 05
Optical Readout

A camera observes the reaction wells and detects the colorimetric result of amplification. Results are reported back to the master node, which interprets them according to the disk's instruction set.

N — 06
Microfluidic Cartridge

The disposable disk that holds reagents and defines the routing geometry. Different disk types map to different protocols — the master node identifies the disk and loads the matching instruction sequence.
